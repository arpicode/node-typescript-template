# node-typescript-template

Minimal `Node.js / TypeScript` project setup with:

- `ESLint` with typed `typescript-eslint` rules
- `prettier`
- `vitest`

Requirements:

- `Node.js` `20.11+`
- `pnpm`

Available scripts:

- `pnpm dev`
- `pnpm build`
- `pnpm typecheck`
- `pnpm lint`
- `pnpm format`
- `pnpm test`
- `pnpm validate`
